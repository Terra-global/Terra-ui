# 🌍 Terra UI: Climate Adaptation Dashboard

> The official frontend orchestrator for the **Terra Oracle Protocol**. Turning satellite climate data into expert agronomic advice with one click.

Live Demo: [https://terra-ui.vercel.app](https://github.com/Terra-global/Terra-ui) (Pending Deployment)

---

## 🛠️ The Tech Stack

- **Core:** React 18 + TypeScript + Vite
- **Data Layer:** [@terra-oracle/terra-oracle](https://www.npmjs.com/package/@terra-oracle/terra-oracle)
- **Cognitive Layer:** Google Gemini 1.5 Flash / Gemini 3
- **Styling:** Custom "Agri-Modern" Design System
- **Icons:** Lucide React

---

## 🚀 Features

- **One-Click Oracle Consulting**: Automatically resolves location and fetches satellite weather.
- **AI Expert Advisory**: Real-time agronomic survival strategies generated by Gemini AI.
- **Multi-tier Geolocation**: GPS-first strategy with silent IP-based fallback.
- **Embedded Documentation**: Full developer guide integrated directly into the dashboard.
- **Zero-Key Architecture**: Local environment variables for AI keys, ensuring decentralized security.

## 💻 Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/Terra-global/Terra-ui.git
cd terra
npm install
```

### 2. Configure Environment
Create a `.env` file in the root:
```env
VITE_GEMINI_API_KEY=your_google_ai_studio_key
```

### 3. Run Development
```bash
npm run dev
```

---

## 📖 Architecture

Terra UI acts as the **Orchestrator** between three distinct layers:

1. **The User Layer**: Collects crop data and requests analysis.
2. **The Data Layer (Oracle)**: Fetches raw climate data from satellite fusion (Open-Meteo).
3. **The Cognitive Layer (AI)**: Injects the Oracle's "Fact Sheet" into Gemini to provide a human-readable advisory.
